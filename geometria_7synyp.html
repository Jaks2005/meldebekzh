Теориялық материал

1. Үшбұрыштың түрлері

Үшбұрыш — бір түзуде жатпайтын үш нүктені жалғастыратын үш кесіндіден тұратын геометриялық фигура.

Қабырғаларына қарай:

Теңқабырғалы
a = b = c, барлық бұрышы 60°
Теңбүйірлі
a = b ≠ c, негіздегі бұрыштар тең
Әртүрлі қабырғалы
a ≠ b ≠ c

Бұрыштарына қарай:

∠
Сүйір бұрышты
Барлық бұрышы < 90°
⊾
Тікбұрышты
Бір бұрышы = 90°
⬡
Доғал бұрышты
Бір бұрышы > 90°

2. Биссектриса, Медиана, Биіктік, Орта сызық

Биссектриса
Бұрышты тең екіге бөліп, қарсы қабырғаға жеткен кесінді
Медиана
Төбеден қарсы қабырғаның ортасына жүргізілген кесінді
Биіктік
Төбеден қарсы қабырғаға перпендикуляр
Орта сызық
Екі қабырғаның ортасын жалғайтын кесінді
Орта сызық теоремасы: MN ∥ BC, MN = ½ · BC
  • 3 медиана → ауырлық орталығы (2:1 қатынасында бөледі)
  • 3 биіктік → ортоцентр
  • 3 биссектриса → ішкі шеңбер орталығы
  • 3 орта перпендикуляр → сырттан шеңбер орталығы

3. Теңқабырғалы, Теңбүйірлі, Тікбұрышты үшбұрыш элементтері

Теңқабырғалы (a = b = c):

  • ∠A = ∠B = ∠C = 60°
  • Медиана = Биссектриса = Биіктік (бір сызық)
  • Биіктігі: h = a√3/2

Теңбүйірлі (a = c, b — негіз):

  • Негіздегі бұрыштар тең: ∠A = ∠C
  • Негізге жүргізілген медиана = биссектриса = биіктік

Тікбұрышты (∠C = 90°):

  • Гипотенуза (c) — тік бұрышқа қарама-қарсы жатқан ең ұзын қабырға
  • Катеттер (a, b) — тік бұрышты жасайтын қабырғалар
  • Тік бұрышқа жүргізілген медиана = c/2
Пифагор теоремасы: a² + b² = c²

4. Сүйір, Доғал, Тікбұрышты үшбұрыштардың биіктіктерін салыстыру

Үшбұрыш түріБиіктіктердің орныОртоцентр
∠ Сүйір бұрыштыҮшеуі де ішіндеҮшбұрыш ішінде
⊾ ТікбұрыштыЕкі катет — биіктік, үшіншісі ішіндеТік бұрыш төбесінде
⬡ Доғал бұрыштыЕкі биіктік сырттаҮшбұрыш сыртында

5. Үшбұрыштар теңдігінің белгілері

I
ҚЖҚ белгісі
Екі қабырға және олардың аралық бұрышы тең
II
ҚБҚ белгісі
Бір қабырға және екі жағындағы бұрыштар тең
III
ҚҚҚ белгісі
Үш қабырғасы сәйкес тең
△ABC ≅ △A₁B₁C₁ ⟺ белгілердің бірі орындалса

Тікбұрышты үшбұрыш үшін қосымша белгілер:

  • ГК — гипотенуза мен катет тең
  • ГБ — гипотенуза мен жедел бұрыш тең
  • КК — екі катет тең
  • КБ — катет пен іргелес жедел бұрыш тең

🔻 6. Теңбүйірлі үшбұрыш: қасиеттері және белгілері

Анықтама: Екі қабырғасы (бүйірлері) тең үшбұрыш.

Қасиеттері:

  • Негіздегі бұрыштары тең: ∠B = ∠C
  • Негізге жүргізілген биссектриса = медиана = биіктік — бір сызық
  • Осы сызық — үшбұрыштың симметрия осі

Белгілері (үшбұрыш теңбүйірлі болады, егер):

  • Екі бұрышы тең болса: ∠B = ∠C ⟹ AB = AC
  • Медиана биіктікпен немесе биссектрисамен сәйкес келсе
AB = AC ⟺ ∠ABC = ∠ACB

7. Бұрыштар қосындысы. Сыртқы бұрыш

∠A + ∠B + ∠C = 180°

Сыртқы бұрыш теоремасы: Үшбұрыштың сыртқы бұрышы іргелес емес екі ішкі бұрыштың қосындысына тең.

∠BCD = ∠A + ∠B

Салдар: Сыртқы бұрыш кез келген ішкі іргелес емес бұрыштан үлкен.

8. Үшбұрыш теңсіздігі

a + b > c    a + c > b    b + c > a

Мағынасы: Кез келген қабырға қалған екі қабырғаның қосындысынан кіші.

Үш кесіндіТексеруНәтиже
3, 4, 53+4=7 > 5 ✓Үшбұрыш болады
2, 3, 62+3=5 < 6 ✗Болмайды
5, 7, 95+7=12 > 9 ✓Үшбұрыш болады
1, 1, 31+1=2 < 3 ✗Болмайды

Салдар: Үлкен бұрышқа үлкен қабырға қарама-қарсы жатады.

📐 9. Тікбұрышты үшбұрыштың қасиеттері және теңдік белгілері

  • Жедел бұрыштарының қосындысы: α + β = 90°
  • Тік бұрышқа жүргізілген медиана = гипотенузаның жартысы: m_c = c/2
  • 30°-60°-90° үшбұрышта: кіші катет = c/2, үлкен катет = c√3/2
  • 45°-45°-90° үшбұрышта: катеттер тең, c = a√2
a² + b² = c²  |  sin α = a/c  |  cos α = b/c  |  tan α = a/b

Практикалық бөлім

GeoGebra — интерактивті салу

GeoGebra-да тікелей салу, өлшеу және тексеру жасаңыз:

Кеңес: «Үшбұрыш», «Бұрышты өлшеу», «Қашықтықты өлшеу» және «Медиана» құралдарын пайдаланыңыз.

Тапсырма 1

«Шешімін көру» батырмасын басып жауапты тексеріңіз.

1а

x табыңыз: Теңбүйірлі үшбұрышта екі бұрыш 72° тең. x = ?

ТеңбүйірліБұрыш
x + 72 + 72 = 180° → 2(72) = 144 → x = 180 − 144 = 36°
1б

x табыңыз: Үшбұрышта P = 70°, екі бұрыш x тең. x = ?

Теңбүйірлі
70 + x + x = 180 → 2x = 110 → x = 55°
1в

x табыңыз: Үшбұрышта x° және 2x° бұрыштары бар, сыртқы бұрыш белгіленген.

Сыртқы бұрыш
Сыртқы бұрыш = x + 2x = 3x. x + 2x + үш. бұр = 180. x = 30°
1д

x табыңыз: Үшбұрышта ∠A = 120° — сыртқы бұрыш, теңбүйірлі. x = ?

Сыртқы бұрышТеңбүйірлі
Сыртқы бұрыш = x + x → 120 = 2x → x = 60°
1е

x табыңыз: Теңбүйірлі үшбұрышта ∠D = 65°, тең белгілер берілген. x = ?

Теңбүйірлі
x + x + 65 = 180 → 2x = 115 → x = 57.5°
2а

x табыңыз: Теңбүйірлі үшбұрышта ∠B = ∠C = 75°, бүйір = 16 см. x = ?

ТеңбүйірліҚабырға
Тең бұрыштарға қарсы жатқан қабырғалар тең → x = 16 см
3а

Жіктеңіз: ∠A = 60°, белгіленген үшбұрыш. Қандай үшбұрыш?

Жіктеу
Үш бұрышы да 60° → Теңқабырғалы (Equilateral)
3г

Жіктеңіз: △JKL-да ∠J = 75°, ∠K = 30°. Қандай үшбұрыш?

Жіктеу
∠L = 180 − 75 − 30 = 75°. ∠J = ∠L = 75° → Теңбүйірлі (Isosceles)
4

x табыңыз: △ABC-да ∠A = x°, ∠B = (x+24)°, ∠C = 52°. x = ? Үшбұрыш қандай?

ТеңбүйірліЕсептеу
x + (x+24) + 52 = 180 → 2x + 76 = 180 → 2x = 104 → x = 52°. ∠A = ∠C = 52° → Теңбүйірлі, AB = BC
Ex4C #4а

a табыңыз: Үшбұрышта бұрыштар a°, 4a°, 4a°.

Бұрыш
a + 4a + 4a = 180 → 9a = 180 → a = 20°. Бұрыштар: 20°, 80°, 80°
Ex4C #4б

b табыңыз: Бұрыштар: b°, (b+5)°, (b−5)°.

Бұрыш
b + (b+5) + (b−5) = 180 → 3b = 180 → b = 60°. Бұрыштар: 60°, 65°, 55°
Ex4C #4в

c және d табыңыз: Үшбұрышта 48°, 76°, c°, сыртқы бұрыш d°.

Сыртқы бұрыш
c = 180 − 48 − 76 = 56°. d = 48 + 76 = 124° (сыртқы бұрыш теоремасы)
Ex4C #5

Бұрыштарды табыңыз: Үшбұрыш бұрыштары x°, (x−12)° және (2x+6)°.

Теңдеу
x + (x−12) + (2x+6) = 180 → 4x − 6 = 180 → 4x = 186 → x = 46.5°. Бұрыштар: 46.5°, 34.5°, 99°
Пифагор 1

Гипотенузаны табыңыз: Катеттері 3 см және 2 см.

Пифагор
x² = 3² + 2² = 9 + 4 = 13 → x = √13 ≈ 3.61 см
Пифагор 2

Катетті табыңыз: Гипотенуза 6 см, бір катет 5 см. Екінші катет = ?

Пифагор
x² + 5² = 6² → x² = 36 − 25 = 11 → x = √11 ≈ 3.32 см
Лувр

STEM: Лувр пирамидасы: табан қабырғасы 35 м, биіктігі 20.6 м. Жанама қырын тап.

STEMПифагор
Жарты табан = 17.5 м. l² = 17.5² + 20.6² = 306.25 + 424.36 = 730.61 → l ≈ 27.03 м

Тапсырмалар 2

Дәлелдеу есептері — ең маңызды тақырып.

§3 #1

Бет 44: AB және CD кесінділері O нүктесінде қиылысады, O нүктесі екеуінің де ортасы. AC = 10 м болса, BD = ?

ТеңдікSAS
△AOC және △BOD: AO=OB, CO=OD, ∠AOC=∠BOD (төбелес бұрыштар) → SAS → △AOC≅△BOD → BD = AC = 10 м
§3 #2

Бет 44: AB кесіндісінің O ортасы арқылы AB-ға перпендикуляр түзу жүргізілген. XA = XB екенін дәлелде.

ДәлелдеуSAS
△XOA және △XOB: OA=OB (O ортасы), XO ортақ, ∠XOA=∠XOB=90° → SAS → XA = XB ✓
§3 #4

Бет 44: CD=AC, CE=BC арқылы ED = AB екенін дәлелде.

ДәлелдеуSAS
△ACB және △DCE: AC=DC, BC=EC (шарт), ∠ACB=∠DCE (төбелес) → SAS → ED = AB ✓
Бет 45 #9

Теңбүйірлі үшбұрыштың периметрі 1 м, табаны 0.4 м. Бүйір қабырғасын тап.

ТеңбүйірліПериметр
2a + 0.4 = 1 → 2a = 0.6 → a = 0.3 м
Бет 45 #10

Теңбүйірлі үшбұрыштың периметрі 7.5 м, бүйір қабырғасы 2 м. Табанын тап.

ТеңбүйірліПериметр
2(2) + c = 7.5 → 4 + c = 7.5 → c = 3.5 м. Тексеру: 2+2=4 > 3.5 ✓
Бет 45 #11

Периметр 15.6 м, теңбүйірлі үшбұрыш. 1) Табан бүйірден 3 м-ге аз. 2) Табан бүйірден 3 м-ге артық.

ТеңбүйірліТеңдеу
1) c = a−3: 2a+(a−3)=15.6 → 3a=18.6 → a=6.2 м, c=3.2 м. 2) c=a+3: 2a+(a+3)=15.6 → 3a=12.6 → a=4.2 м, c=7.2 м
Бет 64 #29

△ABC-да A мен B-дан биссектрисалар D нүктесінде қиылысты. ∠ADB=? 1) ∠A=50°, ∠B=100°; 2) ∠C=130°

БиссектрисаБұрыш
Формула: ∠ADB = 90° + ∠C/2. 1) ∠C=30° → ∠ADB=105°. 2) ∠ADB=90+65=155°
Бет 64 #32

Теңбүйірлі үшбұрыштың бір сыртқы бұрышы 70°. Бұрыштарын тап.

ТеңбүйірліСыртқы бұрыш
Ішкі бұрыш = 110° (төбе). Табан бұрышы = (180−110)/2 = 35°. Жауап: 110°, 35°, 35°
Бет 64 #33

Екі сыртқы бұрыш 120° және 150°. Үшбұрыш бұрыштарын тап.

Сыртқы бұрыш
∠A=60°, ∠B=30°, ∠C=90°. Бұрыштар: 60°, 30°, 90°
Бет 64 #38

A және B-дағы сыртқы бұрыштар қосындысы 240°. ∠C = ?

Сыртқы бұрыш
(180−∠A)+(180−∠B)=240 → ∠A+∠B=120 → ∠C=60°
Бет 64 #41

Тікбұрышты △ABC-да BD биіктігі жүргізілген. ∠CBD=? 1) ∠A=20°; 2) ∠A=65°; 3) ∠A=α

ТікбұрыштыБиіктік
∠CBD=∠A. 1) 20°. 2) 65°. 3) α

STEM Есептері

Архитектура STEM

Есеп 1: Үйдің шатыры 6 м ені, биіктігі 2.4 м. Шатыр жапқышының ұзындығы?

Жарты ені = 3 м. l = √(3² + 2.4²) = √14.76 ≈ 3.84 м
Табиғат STEM

Есеп 2: Ағаш 60° бұрышпен 8 м сынды. Сынуға дейінгі биіктік?

Тік бөлік = 8·sin60° ≈ 6.93 м. Жалпы ≈ 14.93 м
Физика STEM

Есеп 3: Зымыран 30° бұрышпен 500 м горизонталь жүрді. Нақты ұшу қашықтығы?

l = 500/cos30° ≈ 577 м
География STEM

Есеп 4: Үш қала: 50 км, 70 км, 90 км. Үшбұрыш теңсіздігін тексер.

50+70=120>90 ✓, 50+90=140>70 ✓, 70+90=160>50 ✓ → Үшбұрыш болады!
IT STEM

Есеп 5: A(0,0), B(6,0), C(3,4). Периметр мен ауданды есепте.

AB=6, BC=5, AC=5. P=16 бірлік. S=12 бірлік²
Инженерия STEM

Есеп 6: Катеттері 12 м және 16 м. Жабын ұзындығы?

c = √(144+256) = √400 = 20 м
